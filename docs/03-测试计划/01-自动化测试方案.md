## 五、测试数据管理

### @Transactional 自动回滚
所有测试使用 `@SpringBootTest` + 真实数据库（H2 内存），每个测试类 `@BeforeEach` 清空相关表，保证独立性。

### 角色初始化（关键）
测试环境中 `DataInitializer` 被 `@Profile("!test")` 禁用，测试需要手动创建角色：
```java
SysRole adminRole = new SysRole();
adminRole.setRoleName("系统管理员");
adminRole.setRoleCode("ADMIN");
adminRole.setDescription("管理员");
SysRole savedRole = roleRepository.save(adminRole);
// 注册用户后分配角色
userRepository.findByUsername("admin").ifPresent(u -> {
    u.setRoles(Set.of(savedRole));
    userRepository.save(u);
});
```
